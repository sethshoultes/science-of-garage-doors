Interactive educational presentations about garage door science. Six Gutenberg blocks for embedding on any WordPress page or post.

== Description ==

A Plus Garage Door Science bundles six interactive educational presentations as embeddable Gutenberg blocks:

1. **Science of Garage Doors** — Interactive 3D explorer with six animated Three.js scenes covering mechanics, springs, drive types, motors, and smart home integration.
2. **Science of Garage Door Springs** — Torsion spring physics, safety data, energy comparisons, and an interactive winding simulation.
3. **Rolling Steel Doors Lab** — Engineering and mechanics of rolling steel/coiling doors.
4. **Garage Door ROI Calculator** — Interactive return-on-investment calculator for new garage door installations.
5. **Energy Efficiency Lab** — Insulation types, R-values, and energy savings analysis.
6. **Spring Fatigue & Cold Weather** — How temperature and fatigue cycles cause spring failures.

Each presentation is also available as a shortcode for use in classic editor or widgets.

== Gutenberg Blocks ==

All blocks appear under the **Embeds** category in the block inserter:

* Science of Garage Doors
* Science of Garage Door Springs
* Rolling Steel Doors Lab
* Garage Door ROI Calculator
* Energy Efficiency Lab
* Spring Fatigue & Cold Weather

Each block has a configurable **Height** setting in the sidebar.

== Shortcodes ==

* `[aplus_garage_door_science height="900px"]`
* `[aplus_spring_science height="800px"]`
* `[aplus_rolling_steel height="800px"]`
* `[aplus_garage_door_roi height="800px"]`
* `[aplus_energy_efficiency height="800px"]`
* `[aplus_spring_fatigue height="800px"]`

== Installation ==

1. Download the plugin zip from the [GitHub Releases page](https://github.com/sethshoultes/science-of-garage-doors/releases/latest)
2. In WordPress, go to Plugins → Add New → Upload Plugin
3. Upload the zip file and activate
4. Add any block by searching for its name in the block inserter

== Frequently Asked Questions ==

= Can I customize the height? =

Yes. In the Gutenberg block, use the Height setting in the sidebar. With shortcodes, use the height attribute.

= Does it work with page builders? =

Yes. Use any of the shortcodes in any page builder that supports shortcodes.

= Does the plugin auto-update? =

Yes. The plugin checks GitHub Releases for new versions and supports one-click updates from wp-admin.
